# laba6
---
Репозиторий содержит три консольных приложения на Java для отработки навыков работы с коллекциями (`Map`, `List`), дженериками и алгоритмами сортировки данных.

## Состав проекта
--- 

### 1. TopWords.java
Программа для частотного анализа текста. Считывает содержимое файла, подсчитывает количество повторений каждого слова и выводит топ-10 самых популярных слов в порядке убывания частоты. Для очистки слов от знаков препинания используются регулярные выражения.

**Запуск:**
```bash
javac TopWords.java
java TopWords
```

### 2. Stack.java
Реализация собственной структуры данных — обобщенного (Generic) стека на основе массива. Поддерживает работу с любыми типами данных и реализует базовые операции стека: добавление (`push`), удаление с возвратом (`pop`) и просмотр верхнего элемента (`peek`).

**Запуск:**
```bash
javac Stack.java
java Stack
```

### 3. ShopApp.java
Программа для автоматизации учета продаж в магазине. Использует коллекцию `HashMap` для хранения информации о проданных товарах и их количестве. Позволяет добавлять новые продажи, выводить полный список реализованной продукции, рассчитывать общую выручку и определять самый популярный товар.

**Запуск:**
```bash
javac ShopApp.java
java ShopApp
```

---

### Особенности реализации
*   **Дженерики**: В задании со стеком реализован принцип параметризации типов для обеспечения безопасности на этапе компиляции.
*   **Сортировка**: В задании с анализом текста используется интерфейс `Comparator` для кастомной сортировки списка записей `Map.Entry`.
*   **Обработка ошибок**: Программы включают в себя блоки `try-catch` для обработки исключений, связанных с отсутствием файлов или некорректным состоянием структур данных (пустой стек).
